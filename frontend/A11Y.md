# Accessibility Testing

Simple accessibility testing with pa11y using Chromium.

## Setup (one-time)

```bash
# Install Chrome for puppeteer (run once)
npx puppeteer browsers install chrome
```

## Usage

```bash
# Test all pages
pnpm run test:a11y

# Test a single page
pnpm run test:a11y:single http://localhost:3000
```

## Configuration

- Edit `.pa11yrc.js` to change settings
- Edit `test-a11y.sh` to change which pages are tested
- Tests follow WCAG 2.1 AA standards
